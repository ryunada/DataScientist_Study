# 스케일링의 종류
|   |       종류      |                  설명                | 
|:---:|:----------------:|:-------------------------------------:|
| 1. | StandardScaler | 기본 스케일링, 평균과 표준편차 사용|
| 2. | MinMaxScaler | 최대/최소값이 각각 1, 0이 되도록 스케일링 |
| 3. | MaxAbsScaler | 최대절대값과 0이 각각 1, 0이 되도록 스케일링 |
| 4. | RobustScaler | 중앙값(median)과 IQR(Interquartile range) 사용, 아웃라이어의 영향을 최소화 |
